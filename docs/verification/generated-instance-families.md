# Generated instance families

Planned problem-shape families:

- tiny known-answer LPs;
- random feasible bounded LPs;
- primal/dual certificate LPs;
- network-flow-like LPs;
- transportation LPs;
- block-angular LPs;
- scenario-tree LPs;
- degenerate LPs;
- ill-conditioned LPs;
- wide sparse LPs;
- tall sparse LPs.

Every generated instance should record the generator name, seed, size
parameters, scaling parameters, and expected evidence when known.
