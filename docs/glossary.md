# Glossary

- **Adapter**: Module that invokes a solver and normalizes its output.
- **Canonical LP**: The internal LP representation used by the harness.
- **G0**: Design-control scaffold gate.
- **Harness**: Test and benchmark runner that executes solver adapters.
- **Native-targeted code**: Code expected to be reachable from GraalVM Native
  Image executable or shared-library builds.
- **Problem family**: A generated class of LP instances with parameterized shape.
- **Tolerance profile**: Named set of numerical thresholds for validation.
